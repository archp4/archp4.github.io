Novel Audio Generator

This project features a real-time data scarping of novel then audio generation that fetches data from Novel Cool website via a custom API made in python. Users can request audio files for their favorite novel chapters instantly. Using python FASTAPI farmework, the API converts text content into high-quality audio files on demand. The emphasis on real-time functionality ensures users receive prompt and up-to-date audio renditions. This innovation caters to diverse preferences, enhancing accessibility and providing a dynamic reading experience through audio format.
